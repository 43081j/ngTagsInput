Documentation

tagsInput (directive in module tags-input)

Description

ngTagsInput is an Angular directive that renders an input control with tag editing support.

Usage

{% highlight html %} {% endhighlight %}

Parameters

Param Type Description Defaults to
ngModel (required) string Assignable angular expression to data-bind to. –
ngClass string CSS class to style the control. –
tabindex number Tab index for the new tag input box. –
placeholder string Text placeholder for the new tag input box. Add a tag
removeTagSymbol string Character to remove an existing tag. ×
minLength number Minimum length for a new tag. 3
maxLength number Maximum length for a new tag. The placeholder length or the min-length value, whichever is greater.
addOnEnter boolean Flag indicating that a new tag will be added on pressing the ENTER key. true
addOnSpace boolean Flag indicating that a new tag will be added on pressing the SPACE key. false
addOnComma boolean Flag indicating that a new tag will be added on pressing the COMMA key. true
allowedTagsPattern string Regular expression that determines the pattern used to validate a tag. ^[a-zA-Z0-9\s]+$*
enableEditingLastTag string Flag indicating that the last tag will be moved back into the new tag input box instead of being removed when the backspace key is pressed and the input box is empty. false
